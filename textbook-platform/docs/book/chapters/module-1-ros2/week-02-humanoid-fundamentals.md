# Week 2: Fundamentals of Humanoid Robotics

## Learning Objectives

1. Understand humanoid robot mechanical structure and degrees of freedom
2. Identify key actuator types (servo motors, linear actuators)
3. Explain sensor arrays used in humanoid platforms
4. Describe control hierarchies (low-level, mid-level, high-level)
5. Analyze trade-offs in humanoid robot design
